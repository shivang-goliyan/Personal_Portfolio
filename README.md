# Professional Portfolio - Shivang Goliyan

A modern, responsive, single-page portfolio website built with HTML, Tailwind CSS, and JavaScript.

## Features
- **Dark/Light Mode**: Persists user preference.
- **Project Filtering**: Categorize projects by type (Web, SaaS, Tools).
- **Responsive Design**: Mobile-friendly with a custom hamburger menu.
- **Animations**: Smooth scrolling and hover effects.

## Setup
1. Clone the repository.
2. Open `index.html` in your browser.

## Deployment
This project is optimized for static hosting (Vercel, Netlify, GitHub Pages).
Just simply deploy the repository and strict valid HTML will be served.
